# Маркетинговая стратегия Ozonbox

## Позиционирование

### УТП (Уникальное торговое предложение)

- **Российское производство** — локализация, независимость от импорта
- **Патент РФ №2735850** — холодная генерация озона (до 60°C vs 150°C у конкурентов)
- **5 лет гарантии** — срок службы разрядников 10 лет
- **Без расходных материалов** — не требует замены элементов
- **Комплексные решения** — оборудование + АСУ ТП

## Контент-маркетинг

### SEO-стратегия

**Приоритетные направления:**

1. **Отраслевые статьи** — проблемы и решения в каждой отрасли
2. **Технические материалы** — как работает озонирование, эффективность
3. **Кейсы** — реальные проекты с результатами
4. **Сравнения** — Ozonbox vs конкуренты, химическая обработка vs озон

**Структура контента:**

```
Тема → Статья → Лендинг → КП
```

### Виды контента

| Тип | Примеры | Канал |
|-----|---------|-------|
| **Образовательный** | "Как выбрать озонатор", "Принцип работы озона" | Блог, SEO |
| **Отраслевой** | "Озонирование зерна — эффективность", "Птичий грипп профилактика" | Блог, соцсети |
| **Кейсы** | "Кейс: Птицефабрика Ульяновск", "Овощехранилище на 5000 тонн" | Блог, КП |
| **Технический** | "Технические характеристики", "Монтаж и пусконаладка" | Сайт, документация |
| **Продающий** | "Рассчитать стоимость", "Подбор оборудования" | Лендинги |

## Каналы продвижения

### Сайт и SEO

- **ozonbox.pro** — основной сайт
- **Лендинги по отраслям** — специализированные посадочные страницы
- **Блог** — статьи по отраслям и технологиям
- **Каталог** — продукты с характеристиками

### Социальные сети

- **YouTube** — видео-обзоры, кейсы, вебинары
- **ВКонтакте** — профессиональное сообщество
- **Telegram** — новости, экспертный канал

### Партнёрская сеть

- **20+ городов** — региональные партнёры
- **Совместные КП** — проектные продажи
- **Обучающие мероприятия** — семинары для партнёров

### Выставки и мероприятия

- **Агропромышленные выставки** — Agrolearning, региональные
- **Профильные конференции** — ветеринария, хранение
