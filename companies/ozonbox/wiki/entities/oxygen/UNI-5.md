# UNI-5

## Цена
390 000 руб. (без НДС)

## Описание

Промышленная кислородная озонаторная установка серии UNI со встроенным источником кислорода.

## Характеристики

| Параметр | Значение |
|----------|----------|
| Производительность озона | 10 г/ч |
| Напряжение питания | 220В |
| Мощность | 450 Вт |
| Исполнение | 1 шкаф, напольное |
| Компоновка | Моноблок |
| Охлаждение | Воздушное |
| Источник газа | Встроенный кислородный блок |
| Управление | PLC / промышленная автоматика |
| Габариты (Д×Ш×В) | 1400×600×400 мм |
| Масса | 78 кг |
| Гарантия | 5 лет |

## Особенности

- Встроенный источник кислорода
- Озоностойкий газовый тракт
- Резервирование генераторной камеры
- Протокол Modbus
- Интеграция с Oz Control

## Комплектация

- Защита газовой магистрали
- Внешнее управление (Modbus)
- Эжектор
- Датчик потока/протока
- Инструкция
- Ударопрочный транспортировочный ящик
