# UNI-400 — Кислородный озонатор

**Цена:** 1 790 000 руб. (без НДС)

**Артикул:** UNI400

**Категория:** [[Кислородные озонаторы и станции|Oxygen]] — Кислородные озонаторы и станции

**Гарантия:** 5 лет

---

## Описание

Промышленная кислородная озонаторная установка серии UNI со встроенным источником кислорода, системой управления на базе промышленного контроллера и резервированием генераторной камеры.

---

## Характеристики

| Параметр | Значение |
|----------|----------|
| Производительность озона | 400 г/ч |
| Напряжение питания | 380В |
| Мощность | 10200-11000 Вт |
| Исполнение | 1 шкаф, напольное |
| Компоновка | Моноблок |
| Охлаждение | Воздушное |
| Резервирование | 8 резервных генераторных камер |
| Источник газа | Встроенный кислородный блок |
| Управление | PLC / промышленная автоматика |
| Материал корпуса | Нержавеющая сталь / сталь с порошковым окрасом |
| Габариты (Д×Ш×В) | 2000×1200×800 мм |
| Габариты в упаковке | 2140×1290×850 мм |
| Масса нетто / брутто | 520 / 624 кг |
| Срок службы | 30 000 часов |

## Комплектация

- Защита газовой магистрали
- Внешнее управление (Modbus)
- Эжектор
- Датчик потока/протока
- Инструкция
- Ударопрочный транспортировочный ящик
