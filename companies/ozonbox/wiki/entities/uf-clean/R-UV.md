# Ozonbox R-UV — УФ подвесные рециркуляторы

**Артикул:** RUV

**Категория:** [[УФ-оборудование для очистки воздуха|UF-Clean]] — УФ-оборудование для очистки воздуха

---

## Описание

Предназначены для рециркуляционной (без приточки) стерилизации воздуха внутри помещения. Устройство забирает воздух из объема, стерилизует его и выбрасывает обратно. Важный и одновременно часто игнорируемый элемент для создания безопасной среды на пищевом/птицеводческом производстве.

**Уникальность:**
- Контроль — полностью интегрируется с системой Oz Control, позволяя контролировать данные с устройства, управлять с корреляцией на режим работы производства, остаток ресурса ламп
- Готовность из коробки — не требует сложных монтажных манипуляций и подключений
- Система закрытого типа, не создаёт облучения для глаз
- Полный «bolt on» монтажный комплект в составе оборудования
