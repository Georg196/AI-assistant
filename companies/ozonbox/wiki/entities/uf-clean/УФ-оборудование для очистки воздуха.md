# УФ-оборудование для очистки воздуха

## Описание

Системы УФ-очистки воздуха для стерилизации приточного воздуха и рециркуляции воздуха в помещениях.

## Товары

| Модель | Описание |
|--------|----------|
| [I-UV](I-UV.md) | УФ приточные стерилизаторы |
| [R-UV](R-UV.md) | УФ подвесные рециркуляторы |
| [SMI-UV](SMI-UV.md) | Интегрируемые блок-модули УФ |
| [SMI-UVO](SMI-UVO.md) | Интегрируемые блок-модули УФ + озонирование |

## Особенности

- Стерилизация воздуха УФ-излучением
- Работа в присутствии людей
- Комбинация УФ и озона
- Интеграция в существующие системы вентиляции
