# I-UV — УФ приточные стерилизаторы воздуха

**Цена:** По запросу

**Артикул:** IUV

**Категория:** [[УФ-оборудование для очистки воздуха|UF-Clean]] — УФ-оборудование

---

## Описание

Предназначены для создания микробиологического барьера через попадающий в помещение воздух. Специально подобранный тип УФ источников и конструкция модуля позволяет нейтрализовать патогены в потоковом режиме в присутствии людей.

**Уникальность:**
- Полная интеграция с системой Oz Control
- Мульти модальность — не требует разбора текущей системы вентиляции
- Тип исполнения «bolt on» — стыкующийся модуль
- Система закрытого типа, не создает облучения для глаз
- Не создает сопротивления потоку
- Может использоваться как дополнительная станция нейтрализации озона
