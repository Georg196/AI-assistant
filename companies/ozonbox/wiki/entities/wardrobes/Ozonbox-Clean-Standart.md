# Ozonbox Clean Standart

## Цена
520 000 руб. (без НДС)

## Описание

Промышленный озонирующий шкаф для дезинфекции, дезодорации и сушки одежды, обуви, экипировки, тары и других изделий.

## Характеристики

| Параметр | Значение |
|----------|----------|
| Производительность озона | 40 г/ч |
| Напряжение питания | 220В |
| Мощность | 600 Вт / 2000 Вт (сушка) |
| Объем камеры | 0,71 м³ |
| Габариты камеры (Д×Ш×В) | 810×1511×585 мм |
| Режим сушки | Да |
| Нейтрализатор озона | Да |
| Контроль концентрации озона | 0-100 ppm |
| Материал корпуса | Нержавеющая сталь |
| Габариты (Д×Ш×В) | 1110×600×1970 мм |
| Масса | 229 кг |
| Гарантия | 5 лет |

## Особенности

- Дезинфекция, дезодорация и сушка в одном устройстве
- Сухая обработка без жидких реагентов
- Встроенный нейтрализатор озона
- Программное управление
- Сенсорный LCD дисплей
- Возможность интеграции вендингового модуля

## Применение

- Химчистки и прачечные
- Пищевые производства
- Пожарные части
- Спортивные комплексы
- Авто- и мотоклубы
- Торговые центры
