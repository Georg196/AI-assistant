# Ozonbox AIR-100 Max

## Цена
376 520 руб. (без НДС)

## Серия Air Max

Профессиональная серия для предприятий с высокими требованиями к эффективности и безопасности. Идеальна для объектов с жёсткими санитарными требованиями.

## Характеристики

| Параметр | Значение |
|----------|----------|
| Производительность озона | 100 г/ч |
| Объем помещения (стерилизация) | 1500 м³ |
| Объем помещения (дезинфекция) | 3500 м³ |
| Воздушный поток | 1040×2 м³/ч |
| Напряжение питания | 380В |
| Потребляемая мощность | 1800 Вт |
| Управление | Система Oz Control |
| Материал корпуса | Нержавеющая сталь |
| Габариты (Д×Ш×В) | 1760×335×850 мм |
| Масса | 70 кг |
| Гарантия | 5 лет |

## Особенности серии Max

- Промышленное исполнение для интенсивной эксплуатации
- Высокая надёжность и стабильность обработки
- Защита от перегрева, перегрузки и внештатных режимов
- Усиленная фильтрация для защиты реакторов
- Турбинные вентиляторы для устойчивой подачи воздуха
- Корпус из нержавеющей стали
- Регулировка мощности под задачи объекта
- Пульт управления Oz Control
- Возможность обработки нескольких зон в составе одной системы
- Противопожарная защита

## Комплектация

- Фильтр очистки воздуха (2 шт.)
- Маска защитная озоностойкая
- Инструкция
- Ударопрочный транспортировочный ящик
- Система пожаротушения
- Раздельное управление каждым из каналов
- Раздельное управление мощностью каждого из каналов
- Датчик концентрации озона 0-100ppm
- Датчик влажности

## Меры предосторожности

Нахождение людей и животных в помещении во время озонирования недопустимо.
