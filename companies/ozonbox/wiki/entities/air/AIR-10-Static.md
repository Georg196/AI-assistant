# Ozonbox AIR-10 Static

## Серия Air Static

Стационарные озонаторы для настенного/потолочного монтажа. Идеальны для объектов с высокими требованиями к надёжности.

## Характеристики

| Параметр | Значение |
|----------|----------|
| Производительность озона | 10 г/ч |
| Объем помещения (стерилизация) | 150 м³ |
| Объем помещения (дезинфекция) | 350 м³ |
| Управление | Стационарное крепление |
| Материал корпуса | Нержавеющая сталь |
| Гарантия | 5 лет |

## Особенности серии Static

- Стационарный настенный/потолочный монтаж
- Повышенная надёжность
- Компактное размещение
- Интеграция с системой Oz Control
