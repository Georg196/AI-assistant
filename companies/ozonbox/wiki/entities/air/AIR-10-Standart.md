# Ozonbox AIR-10 Standart

## Цена
39 765 руб. (без НДС)

## Характеристики

| Параметр | Значение |
|----------|----------|
| Производительность озона | 10 г/ч |
| Объем помещения (стерилизация) | 150 м³ |
| Объем помещения (дезинфекция) | 350 м³ |
| Воздушный поток | 290 м³/ч |
| Напряжение питания | 220В |
| Потребляемая мощность | 174 Вт |
| Управление | Механический таймер / Цифровой розеточный таймер |
| Материал корпуса | Нержавеющая сталь |
| Габариты (Д×Ш×В) | 460×200×320 мм |
| Масса | 6,6 кг |
| Срок службы | 30 000 часов |
| Гарантия | 5 лет |

## Особенности

- Патент РФ №2735850 (технология холодной генерации)
- Отсутствие окислов азота
- Корпус из нержавеющей стали с двухстадийным окрасом
- Фильтры тонкой очистки
- Возможность удалённого управления через Oz Control (опционально)
- Возможность увеличения мощности на +20% при подаче свежего воздуха
- Возможность подачи кислорода для увеличения мощности до 50%

## Решаемые задачи

- Дезинфекция
- Дезинсекция
- Дератизация
- Дезодорация
- Детоксикация

## Комплектация

- Цифровой розеточный таймер
- Фильтр очистки воздуха
- Инструкция
- Комплект для озонирования авто
- Ударопрочный транспортировочный ящик

## Меры предосторожности

Нахождение людей и животных в помещении во время озонирования недопустимо.
