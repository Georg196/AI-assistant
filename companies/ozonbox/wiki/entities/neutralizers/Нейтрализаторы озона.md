# Нейтрализаторы озона

## Описание

Каталитические модули для быстрого разложения остаточного озона. Обеспечивают безопасный уровень концентрации для возвращения персонала и сокращение межсменных санитарных окон.

## Задачи

- Быстрая каталитическая нейтрализация остаточного озона
- Обеспечение безопасного уровня концентрации для возвращения персонала
- Сокращение межсменных санитарных окон
- Ускорение возобновления производства после глубокой обработки
- Интеграция в сценарии управления (автоматическое включение в конце озонового цикла)
- Логирование событий
