# DeOxi-4000 — Промышленный нейтрализатор озона

**Цена:** по запросу (под объект)

**Артикул:** DO4000

**Категория:** [[Нейтрализаторы озона|Neutralizers]] — Нейтрализаторы озона

**Гарантия:** 5 лет

---

## Описание

Промышленная система нейтрализации остаточного озона из воздушного потока для объектов большого объёма. Система рассчитана на реальную работу в составе производственных и инженерных контуров.

---

## Характеристики

| Параметр | Значение |
|----------|----------|
| Расчетный объём помещения | до 4000 м³ |
| Количество вентиляторов | 2 |
| Производительность вентиляторов | 2 × 24000 м³/ч |
| Управление | система Oz Control |
| Напряжение питания | 380В |
| Мощность | 9.6-13.2 кВт |
| Материал корпуса | Нержавеющая сталь |
| Габариты (Д×Ш×В) | 3200×1200×2100 мм |
| Габариты в упаковке | 3648×1368×2394 мм |
| Масса нетто / брутто | 1680 / 1860 кг |

## Меры предосторожности

*Нахождение людей, животных в помещении в момент озонирования недопустимо, после включения озонатора покиньте помещение, дождитесь естественного распада или включите нейтрализатор озона после окончания цикла обработки.*
