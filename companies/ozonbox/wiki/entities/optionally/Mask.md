## Описание

Защитная маска с панорамным обзором для надёжной защиты при работе с озоном. Прочный силиконовый корпус с двойным обтюратором позволяет маске одного универсального размера плотно прилегать к лицу.

**Особенности:**
- Панорамный обзор
- Один универсальный размер
- Изготовлена из силикона
- Двойной обтюратор для плотного прилегания
- Два боковых узла для соединения с фильтрами (байонетное крепление)
