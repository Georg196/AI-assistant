# 📁 Компании

Структура для работы с несколькими клиентами/компаниями.

## Структура каждой компании

```
companies/
├── TEMPLATE.md                    # Шаблон для новой компании
├── [company-slug]/
│   ├── company-info.md            # Основная информация
│   ├── projects/                  # Проекты
│   │   └── [project-name].md
│   └── context/                   # Контекст и данные
│       ├── requirements.md
│       ├── branding.md
│       └── technical-stack.md
```

## Структура ozonbox

```
ozonbox/

│   └── TZ.txt             # Техзадание

├── src/                 # Исходники (CSS, JS)
├── dist/                # Скомпилированный CSS

```

## Как добавить компанию

1. Скопировать `TEMPLATE.md` в новую папку
2. Переименовать папку по slug (например, `my-client`)
3. Заполнить `company-info.md`
4. Добавить строку в эту таблицу

## Правила ведения

- `company-info.md` — всегда актуальная информация
- `projects/*.md` — по одному файлу на проект
- После каждой сессии обновлять relevant файлы
