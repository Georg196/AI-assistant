# AI асистент Ozonbox

Персональный AI-ассистент на базе OpenCode для управления проектами и базой знаний Ozonbox.

## Об ассистенте

Я — AI-ассистент, работающий на платформе OpenCode. Мои основные функции:

- Управление проектами и задачами
- Ведение базы знаний в Obsidian (LLM Wiki паттерн)
- Помощь в разработке и документировании
- Анализ и систематизация информации о компании

## О проекте

Проект включает:

### Компании и проекты

- **Ozonbox** — российский производитель промышленного озонаторного оборудования
  - Сайт: https://ozonbox.pro
  - Продукция: озонаторы, системы УФ-очистки, шкафы санитарной обработки
  - Отрасли: пищевые производства, агрохолдинги, зернохранилища, птицеводство

### Wiki (База знаний)

Персональная база знаний в формате LLM Wiki:
- `raw/` — исходные документы и исследования
- `wiki/` — структурированные статьи (entities, concepts, summaries)
- `index.md` — каталог всех страниц
- `log.md` — хронология работы

### Инструменты

- **OpenCode** — AI-платформа для взаимодействия
- **Obsidian** — работа с базой знаний
- **GitHub** — версионирование и синхронизация
