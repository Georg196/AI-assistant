# История изменений проекта Ozonbox Агрохолдинги

## Версия 1.0 - 07.05.2026

### Изменения главного экрана (hero):
- Заголовок h1 вынесен над блоком с max-width: 750px
- Кнопка "Получить расчет" заменена на "Рассчитать" с ссылкой #zv6
- Блок статистики (hero__stats) перенесен отдельно ниже hero
- Добавлен градиентный фон для блока статистики (#088EBC → #1F32BF)
- Картинка на первом экране растянута на всю высоту

### Исправление ссылок на кнопках:
- "Рассчитать" → #zv6
- "Оценить риски" → "Решить проблему" → #popup:myform1
- "Получить карту приоритетов" → "Получить консультацию" → #popup:myform3
- "Сравнить схемы" → "Получить консультацию" → #popup:myform3

- "Подобрать оборудование" → "Консультация" → #popup:myform3

### Адаптивность:
- Добавлены стили для мобильных устройств (768px и 576px)
- Уменьшен padding, font-size на мобильных
- Кнопки теперь вертикально на мобильном
- Статистика в 2 колонки на мобильном

### Исправление глобальных стилей для Тильды:
- Удалены стили с тегов h1-h4, p, a, section
- Добавлены классы .text-h1, .text-h2, .text-h3, .text-h4 для заголовков
- Добавлен класс .text-p для параграфов
- Добавлен класс .text-link для ссылок
- Добавлен класс .section для секций

### Удалено:
- Форма в конце страницы (секция FORM)

### Дополнительно:
- (ОТКАТ) Удалена жирность заголовков в мобильной версии
